## 📌 Project Overview

Personal Finance Manager is a command-line Python application that helps you track your complete financial life in one place — including income, expenses, mutual fund investments, pending payments, money lent to others, and money borrowed — and computes your accurate **net worth** at any time.

The system applies **rule-based AI** principles: every financial operation enforces strict business rules before execution, ensuring data integrity and intelligent decision support.

**Key Features:**
- ✅ Income & expense tracking with timestamps
- 📈 Investment portfolio management (Buy / Sell mutual funds)
- ⏳ Two-stage income model: Pending vs. Confirmed income
- 🤝 Lending tracker (money you've given out)
- 🔴 Borrowing tracker (money you owe)
- 💎 Unified net worth engine (6-dimensional calculation)
- 📤 CSV export for external analysis
- 💾 Zero-dependency, offline, privacy-first JSON persistence

## ⚙️ Setup Instructions

### Step 1: Verify Python is Installed

Open your terminal/command prompt and run:

```bash
python --version
```

You should see something like `Python 3.10.x`. If Python is not installed:

- **Windows / macOS:** Download from [https://www.python.org/downloads/](https://www.python.org/downloads/)
- **Linux (Ubuntu/Debian):**
  ```bash
  sudo apt update && sudo apt install python3
  ```

> **Note:** On some systems, use `python3` instead of `python`. Both work fine.

## 📖 How to Use

When you run the application, you will see:

```
===================================
 💰 ISHANT'S FINANCE MANAGER
===================================
💵 Wallet: ₹0.00
-----------------------------------
1. ➕ Add Income
2. ➖ Add Expense
3. 📈 INVESTMENTS (Buy/Sell)
4. ⏳ PENDING INCOME
5. 📊 Portfolio & Net Worth
6. 📜 History
7. 🟢 LENDING (Give/Receive)
8. 🔴 BORROWING (Take/Return)
9. 📤 Export Data
10. ♻️  RESET
11. ❌ Exit
Option:
```

Type the number for the option you want and press **Enter**.

### Quick Guide

| Option | What it Does |
|---|---|
| **1 – Add Income** | Record salary, pocket money, freelance income, etc. Updates wallet immediately. |
| **2 – Add Expense** | Record any payment or purchase. Deducts from wallet. |
| **3 – Investments** | Buy (invest from wallet) or Sell (redeem back to wallet) mutual funds. |
| **4 – Pending Income** | Add expected income without touching wallet. Later, claim it when received. |
| **5 – Portfolio & Net Worth** | View all your investments and your complete net worth breakdown. |
| **6 – History** | View the full chronological transaction log. |
| **7 – Lending** | Record money you've lent to someone. Track repayments (full or partial). |
| **8 – Borrowing** | Record money you've borrowed. Track money you return (full or partial). |
| **9 – Export** | Save all transactions to `expense_report.csv` in the current folder. |
| **10 – Reset** | Clear monthly transactions and reset wallet to ₹0. Investments are kept safe. |
| **11 – Exit** | Close the application. All data is already auto-saved. |

## 📂 Files Generated by the Application

| File | Description |
|---|---|
| `expense_data_final.json` | Auto-created on first run. Stores all your financial data persistently. |
| `expense_report.csv` | Created when you use Option 9 (Export). Contains your transaction history. |

> **Important:** Do **not** manually edit `expense_data_final.json` unless you know JSON format. Accidental edits could corrupt your data. The app will fall back to a fresh default state if the file is unreadable.

## 🐛 Troubleshooting

| Problem | Solution |
|---|---|
| `python: command not found` | Use `python3` instead, or install Python from python.org |
| `SyntaxError` on running | Ensure you are using Python **3.6 or higher** (not Python 2) |
| CSV export fails with permission error | Close `expense_report.csv` if it is open in Excel, then retry |
| Data appears reset unexpectedly | Check that `expense_data_final.json` is in the same folder as the script |
| Rupee symbol (₹) not displaying | Use a terminal that supports UTF-8 encoding (Windows: use Windows Terminal or enable UTF-8 in CMD) |

**Enable UTF-8 on Windows CMD (if ₹ symbol shows as `?`):**
```cmd
chcp 65001
```

## 🤖 AI/ML Concepts Demonstrated

This project demonstrates the following AI and ML principles:

- **Rule-Based Expert System** – Business rules enforce financial validity (e.g., cannot invest more than wallet balance).
- **State Management** – Persistent JSON-based state mirrors AI agent memory management.
- **Decision Support System** – The net worth engine aggregates multi-dimensional data to support high-level financial decisions.
- **Algorithmic Reasoning** – Partial payment resolution, investment buy/sell logic, and pending income workflows implement multi-step reasoning algorithms.
- **Graceful Degradation** – The system handles corrupt/missing data gracefully, a key requirement in robust AI systems.
